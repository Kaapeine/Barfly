Welcome to BarFly

BarFly manages your Bookmarks Toolbar - it keeps your pinned bookmarks at the front, and automatically organizes recently visited ones after a separator.

This wizard will help you get everything set up in a few quick steps.

🔧

How BarFly works

  • Pinned items — Bookmarks you pin stay at the top, before the separator. Pin/unpin from the right-click context menu.
  • Separator — A divider on the toolbar. Everything before it is pinned; everything after is dynamic.
  • Dynamic list — Bookmarks you visit are automatically added after the separator. Most recently visited bumps to the front.
  • Capacity — Limits how many dynamic bookmarks appear. When full, the oldest drops off.
⚠️
Never delete the separator — Removing it will break BarFly. If accidentally deleted, use the "Rebuild toolbar" button in settings to restore it.
📂

Archive existing bookmarks

Barfly will take over your existing bookmarks toolbar folder so please archive it manually or use the button below to do it. They'll be safely stored in a dated folder under "Other Bookmarks".

Leaving your toolbar as-is is not recomended as BarFly will remove your bookmarks. Use this option only if you were already running BarFly and you're trying to recover toolbar state because the browser cleared it.
